Ручной запуск из командной строки ( запускать cmd из папки с файлом)
python widget.py

Чтобы виджет работал всегда, его нужно добавить в автозагрузку.

    Нажми Win + R, введите shell:startup, нажмите Enter. Откроется папка «Автозагрузка».

    Создайте в этой папке ярлык на ваш файл widget.py.

    Важно: Ярлык должен запускать Python. Для этого в поле «Укажите расположение объекта» ярлыка введите:
    C:\Windows\py.exe "ПОЛНЫЙ_ПУТЬ_К_ФАЙЛУ\widget.py"

Теперь ваш виджет будет запускаться каждый раз при включении компьютера и молча работать в фоне.
